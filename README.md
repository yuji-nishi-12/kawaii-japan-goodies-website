# Japanese Goods E-commerce Website

A modern React.js based e-commerce for an online store selling authentic Japanese goods, such as shoes and apparel, shipping Nationwide from Japan.

## Project Overview

This e-commerce website allows users to browse, search, filter, and purchase Japanese products through a responsive user interface. The website is designed with a red and white color theme, inspired by Japanese aesthetics, and provides a smooth shopping experience from product discovery to checkout.

## Project Inspiration

This project is inspired by a real-world online business and is being developed to help support and streamline the operations of a small online business.

## Features
### Navigation
Responsive Navigation bar component with:
* Home
* Collection (All Products)
* About Us
* Contact
* Search bar
* Account (Login/Sign Up)
* Shopping Cart

### Home Page
* Latest Arrivals
* Best Sellers
* Hero image showcasing the store
Clicking a product redirects users to the Product page

### Collection Page
* View all available products
* Filter products by: Cateogry - Men, Women, Kids, goods
                      Sub-category - Topwear, bottomwear
* Sort products by: Price, Relevancy

### Product Page
* Product details display (e.g., name, price, image, description)
* Select available sizes
* Add items to shopping cart
* View related products

### Shopping Cart Page
* View all added products
* Increase or decrease product quantity
* Proceed to checkout

### Checkout Page
* Input delivery information
* Select payment method
* Complete transaction
* Redirects users to Orders page

### Orders Page
* keeps track of orders

### About Us Page
* Short description of the store
* Store mission and values

### Contact Page
* Store contact details
* Social media links

### Footer
* Copyright information

## Design & UI
